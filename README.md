# 🛒 Siddik Commerce

A full-featured e-commerce web application built with Django, implementing real-world business logic such as product management, order processing, discount systems, secure payments, and multilingual support.

---

## 🚀 Features

- 🔐 User authentication & account management  
- 🛍️ Product listings with categories  
- 🛒 Shopping cart & order management  
- 💸 Multiple discount systems:
  - Flash Sale  
  - Campaign Sale  
  - Product Sale  
- 🌐 Multi-language support (English & Bangla)  
- 💳 SSLCommerz payment integration (Test Mode)
- 🛒 Before-login cart and after-login cart merging
- 💁‍♂️ Product review and rating system with purchase verification
- 🧑‍💼 Admin panel for full control  
- 📱 Responsive design  

---

## 🧰 Tech Stack

- **Backend:** Django  
- **Database:** SQLite / PostgreSQL (configurable)  
- **Payment Gateway:** SSLCommerz  
- **Internationalization:** Django i18n, django-modeltranslation
